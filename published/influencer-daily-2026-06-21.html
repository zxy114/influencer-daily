🔧 用 Codex/Claude Code 搭建 AI 人生顾问

Peter Yang 分享教程：如何将 Codex 或 Claude Code 变成人生和职业决策的 AI 顾问。需要设置 4 个文件：SKILL.md（顾问行为定义）、plan.md（你的目标/原则/精力分配）、learnings.md（经验教训记录）、journal.md（日常反思日志）。

💡 如何应用
用 Claude Code 或 Codex 的 skill 功能，创建这 4 个文件就能跑起来。适合做每周复盘、职业规划、决策辅助。关键是 plan.md 要写得具体——越清晰，AI 给的建议越靠谱。

🤖 Kimi Work Goal Mode — 24/7 长任务自动执行

Kimi 推出 Goal Mode，桌面代理可以 7×24 小时持续运行直到任务完成，专为长周期任务和复杂多步骤工作流设计。

💡 如何应用

🎨 截图 → 可交互模拟器：MiniMax M3 视觉转代码工作流

把黑洞模拟器的截图直接丢进 Kilo Code，切换到 MiniMax M3，prompt 它「把这张截图动画化为可运行的黑洞模拟器」。M3 的视觉理解直接转成可工作的代码。

💡 如何应用
看到任何交互界面截图，都可以尝试「截图 → M3 → 可运行原型」这个链路。适合快速做 demo、原型验证、教学演示。关键 prompt：「把这张截图做成一个能交互的 [X]，保留视觉风格」。
