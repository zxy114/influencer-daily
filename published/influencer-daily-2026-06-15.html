1 Kimi Work — 本地桌面 AI Agent 工作流 ⭐ 9/10
工具工作流
Kimi Work 是一款本地 AI Agent，能在你的桌面上自动完成工作。支持原生 Agent Swarm（最多 300 个子 Agent 并行），搭配 WebBridge 扩展可自动搜索、滚动、点击、输入，最终输出 PPTX、Word、PDF、Excel 等生产级文件。
来源：@Kimi_Moonshot
如何应用：用它来做内容批量生产——输入一个主题，让 Agent 自动搜索素材、整理数据、生成 PPT 汇报或 Excel 数据表。300 个并行 Agent 意味着可以同时处理多个子任务。

3 MiniMax M3 — 开源多模态大模型（可本地部署） ⭐ 8/10
工具
MiniMax M3 开源发布，仅约 428B 参数（激活约 23B），支持 1M token 上下文、原生多模态（文本/图片/视频）、前沿编码能力和长程 Agent 任务。可通过 Unsloth GGUF 量化在 138GB RAM 上本地运行。
来源：@MiniMax_AI
如何应用：有 128GB+ 内存的机器可以用 Unsloth 2-bit GGUF 量化版本本地运行。1M 上下文可以一次性喂入整本书做分析，适合长文档摘要和深度内容研究。
